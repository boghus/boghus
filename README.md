## 🚀 Sobre mí

Desarrollador backend enfocado en integraciones, automatización y construcción de herramientas orientadas a resolver problemas operacionales reales.

Me gusta construir sistemas que:
- Conecten plataformas heterogéneas.
- Automaticen flujos repetitivos.
- Reduzcan la carga operativa diaria.
- Mejoren la eficiencia de workflows complejos.
- Procesen grandes volúmenes de información de forma óptima.

Disfruto especialmente trabajar en arquitecturas orientadas a eventos, procesamiento de datos e integración de IA en entornos de producción.

## ⚙️ Cómo me gusta construir software

- **Simplicidad antes que sobreingeniería:** Diseñar soluciones efectivas sin agregar complejidad innecesaria.
- **Código mantenible a largo plazo:** Escribir software limpio que sea fácil de entender meses después.
- **Automatización primero:** Sustituir el trabajo manual por flujos automatizados y eficientes.
- **Arquitecturas desacopladas:** Fomentar el uso de APIs sólidas y sistemas orientados a eventos.
- **Refactorización y optimización:** Enfoque constante en la depuración y mejora del código existente.
